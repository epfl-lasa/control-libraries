20 # Control Libraries
21 A set of libraries to facilitate the creation of full control loop algorithms,
22 including trajectory planning, kinematics, dynamics and control.
23 
24 Documentation is available at <a href="https://epfl-lasa.github.io/control-libraries/versions/main/">epfl-lasa.github.io/control-libraries</a>
